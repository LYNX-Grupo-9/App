# TemisHub - Mobile App ⚖️

Front-end mobile desenvolvido para o projeto **Lynx** (Grupo 9 - SPTech, 5ª ADS). O TemisHub é uma solução de LegalTech focada em agilizar o acesso à justiça, utilizando Inteligência Artificial para realizar a triagem inicial de casos jurídicos e conectar cidadãos a advogados especialistas.

## 🚀 Fluxo do Aplicativo
O app contempla as 4 etapas principais da jornada do usuário:
1.  **Cadastro:** Registro de novos usuários na plataforma.
2.  **Descrição do Caso:** Interface para detalhamento do problema jurídico e envio de documentos.
3.  **Análise da IA:** Tela de processamento que simula a classificação automática do caso por área do Direito.
4.  **Conexão:** Feed de advogados especialistas que demonstraram interesse no caso publicado.

## 🛠️ Tecnologias Utilizadas
- **React Native**: Framework para desenvolvimento cross-platform.
- **Expo**: Plataforma para facilitação do workflow mobile.
- **React Navigation**: Gerenciamento de rotas e navegação em Stack.
- **Lucide React Native**: Biblioteca de ícones vetoriais.
